# 第一周：基础知识

> 本文规划第一周需要学习的相关内容。



## 学习目标

* 了解图像描述，例如数据集的认识等
* 学习或复习神经网络，含CNN、LSTM等的学习
* 复习、了解 Python、PTorch等编程技术
* 学习Linux操作（或Windows），完成一个基本的图像分类任务



## 本周规划

| 星期 | 学习内容                                   | 参考教程                                                     |
| ---- | ------------------------------------------ | ------------------------------------------------------------ |

| 周二 | 学习 CNN、 LSTM                            | 了解CNN、了解LSTM                                            |
